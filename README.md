## 接口配置：
1、启动后端服务
2、找到项目的config 文件夹，配置不同环境的接口
3、图片上传功能需要配置OSS相关的后端授权接口

## Environment
1、首先安装 node 环境
2、安装npm 工具包
3、命令在项目的根目录下运行

## 安装项目依赖

``` bash
npm install

# 启动开发环境
npm run dev

```
